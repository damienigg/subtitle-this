Cost ladder — settings sorted from "always free" to "configurable cost"

Defaults are the cheapest, no-setup combination. Subtitles work out of the box without any API key. Climb the ladder as you want better quality.

  1. NLLB + audio · totally free, local, ~30 well-supported languages, decent quality. (default — works once you've set Emby URL + key)
  2. DeepL + audio · free 500k chars/month, then paid. Excellent on EU/Asian pairs.
  3. LLM + audio · free if you point at local Ollama / LM Studio. Paid if cloud (Anthropic / OpenAI / …). Best quality on any language pair.
  4. LLM + scene · adds ~20 vision-LLM calls per film for pronouns and gendered agreement. Free if your vision LLM is local.
  5. LLM + cinematic · adds one LLM call per cue with the frame attached. Most calls, most expensive. Free if both LLMs are local.

Sections below are organized so you only fill in what your tier needs. Stay on NLLB + audio → ignore the Translation model and Vision model sections entirely.
